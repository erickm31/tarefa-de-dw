A tag <"pre"> é usada para representar texto pré-formatado. O texto dentro dessa tag é exibido em uma fonte de largura fixa (monoespaçada) e mantém tanto os espaços em branco quanto as quebras de linha exatamente como estão no código fonte.
exemplo:
ENTRADA: 

        <'pre'>
                function greet() {
                    console.log("Hello, world!");
                }
SAIDA: 
        function greet() {
            console.log("Hello, world!");
        }

A tag <"samp"> é usada para representar a saída de um programa de computador. Geralmente, é usada para exibir o resultado de um comando ou a saída de um script.
Exemplo:
<"samp">Output: Hello, world!
Saída: Output: Hello, world!

A tag <'code'> é usada para representar trechos de código dentro de um parágrafo ou linha de texto. Geralmente é usada para destacar palavras-chave, funções ou outras partes importantes do código. Exemplo:

O método <'code'>console.log() é usado para exibir mensagens de saída no console do navegador.

Saída: O método console.log() é usado para exibir mensagens de saída no console do navegador.

A tag <'kbd'> é usada para representar a entrada de teclado do usuário. Ela é frequentemente usada para indicar teclas específicas do teclado que podem ser pressionadas pelo usuário.
Exemplo:

Pressione <'kbd'>Ctrl + <'kbd'>C para copiar o texto.


Saída: Pressione Ctrl + C
